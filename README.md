# crystal-autogui

GUI Automation Library for Crystal

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     crystal-autogui:
       github: hrichharms/crystal-autogui
   ```

2. Run `shards install`
